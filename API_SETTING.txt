RESTfulAPI에 기반하여 작성
로그인 관련 API
기능 분류		Method (행위)	주소 (URI)		설명 및 핵심 로직
로그인		POST			/api/users/login	입력받은 ID와 PWD가 DB 정보와 일치하는지 검증하고 인증 상태를 부여함.
회원가입		POST			/api/users		새로 입력받은 유저 정보(ID, PWD, 닉네임)를 DB에 저장함
내 정보 조회		GET			/api/users/{id}	마이페이지 진입 시, 해당 ID를 가진 유저의 정보(닉네임, 가입일 등)를 불러옴.
정보 수정		PUT			/api/users/{id}	유저가 닉네임이나 비밀번호 등을 변경하면, 해당 ID의 DB 데이터를 업데이트함.
회원 탈퇴		DELETE		/api/users/{id}	DB에서 해당 유저의 정보(그리고 작성한 오답 노트까지)를 완전히 삭제함

오답 노트 관련 API
기능 분류		Method (행위)	주소 (URI)			설명 및 핵심 로직
노트 등록		POST			/api/notes			문제 정보(플랫폼, 번호)와 피드백(틀린 이유, 해결 로직)을 DB에 새롭게 저장
목록 전체 조회	GET			/api/notes			로그인한 유저가 지금까지 작성한 모든 오답 노트의 요약 리스트를 불러옴
노트 상세 조회	GET			/api/notes/{id}		리스트에서 특정 노트를 클릭했을 때(팝업/모달), 해당 노트의 상세한 긴 글(틀린 이유 등)을 쏙 뽑아엄
노트 수정		PUT			/api/notes/{id}		이미 작성해 둔 특정 오답 노트의 내용을 수정해서 DB에 업데이트
노트 삭제		DELETE		/api/notes/{id}		더 이상 필요 없는 특정 오답 노트를 DB에서 삭제.
